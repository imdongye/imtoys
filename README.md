## 개요

'lim'brary is Extendable framework for model and texture viewer with imgui

and there are several toys that use limbrary

-   model simplifier, normal map baker

-   color aware image viewer

-   PBR tester

-   real time hatching tester

-   astar visualizer

## 의존성
